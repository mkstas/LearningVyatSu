  \section*{Цель}
  Цель: Приобрести практические навыки создания и публикации веб-сайта с использованием конструктора Tilda.

  \section*{Задание}
  \begin{itemize}
    \item[--] Изучить теоретический материал
    \item[--] Согласовать тему создаваемого сайта с преподавателем и внести её в таблицу
    \item[--] Разработать сайт по приложенным методическим материалам
    \item[--] Опубликовать сайт и приложить его в отчёт
    \item[--] Сделать отчёт в latex, загрузить его
  \end{itemize}

  \section*{Решение}
  Тема созданного сайта - Лэндинг умного дома. Согласно заданию на сайте были созданы 3 веб-страниц, которые содержали в себе: главную страницу, страница устройств, страница скачивания мобильного приложения.

  Для каждй страницы, в качестве стандартного шаблона используются следующие блоки: шапка с навигацией по сайту, подвал с дублирующей навигацией и копирайтом.
  
  На главной странице (рисунок 1) используются блоки:
  \begin{itemize}
    \item[--] Обложка с заголовком, подзаголовком и фоновым изображением
    \item[--] Блок преимуществ
    \item[--] Блок с формой обратной связи
  \end{itemize}

  На странице устройств используется Zero-block, на котором разамещен список доступных устройств умного дома. Экранная форма представлена на рисунке 2.

  На странице скачивания мобильного приложения также используется Zero-block, на котором размещен заголовок, подзаголовок и ссылки, ведущие на магазин мобильных приложений. Экранная форма представлена на рисунке 3.

  В форме обратной связи существует такие поля, как имя клиента, номер телефона, электронная почта. При отправке формы данные отправляются  в Google таблицу, предаставленная на рисунке 4.
